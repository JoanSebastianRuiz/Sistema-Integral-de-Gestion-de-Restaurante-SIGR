# Sistema-Integral-de-Gesti-n-de-Restaurante-SIGR-
SIGR es una aplicación web desarrollada para optimizar y automatizar los procesos operativos de un restaurante. El sistema permite gestionar pedidos en tiempo real, administrar reservas de clientes, controlar el menú digital, manejar usuarios según roles y generar reportes de ventas y cierre de caja.
